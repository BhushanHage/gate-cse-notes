## How to Use These Notes

1. Start with `introduction.md`.
2. Follow layers in sequence (Physical → Data Link → Network → Transport → Application).
3. Use `08_osi_model_summary.md` and `08_important_differences.md` for quick revision.
4. Revise formulas from `Formula.rmd` and use `networking_study_aid.md` before mock tests.

## Study Guide Overview

- **Concept Building:** Introduction + OSI/TCP-IP model
- **Core Learning:** Layer-wise files (L1 to L7)
- **Exam Revision:** Summary sheets + differences + formulas
- **Final Sprint:** Practice previous-year GATE questions topic by topic
