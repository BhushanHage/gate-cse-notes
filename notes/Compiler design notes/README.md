# Compiler Design Notes

This folder is dedicated to **Compiler Design** study materials for **GATE CSE 2027** preparation.

## Purpose
- Organize compiler design topics in one place.
- Keep revision-oriented notes for quick exam preparation.
- Track topic-wise content and future additions.

## Suggested structure
- `01_introduction.md` - Overview of compilers, phases, and tools.
- `02_lexical_analysis.md` - Tokens, patterns, finite automata.
- `03_syntax_analysis.md` - Parsing techniques, FIRST/FOLLOW, LL/LR.
- `04_semantic_analysis.md` - Type checking and semantic rules.
- `05_intermediate_code_generation.md` - IR forms and translation.
- `06_code_optimization.md` - Basic blocks, data-flow, optimization methods.
- `07_code_generation.md` - Target code generation and register allocation.

Add files as needed while keeping names clear and topic-focused.
