# Introduction to Computer Networks

## 1. What is a Computer Network?
A computer network is a collection of computers, servers, mainframes, network devices, and other hardware components interconnected by communication channels that allow sharing of resources and information.

### In simple words
A network is a group of devices that can talk to each other and share data.

### Why this matters
Without a network, each computer would act alone. A network allows devices to:
- share files
- share printers
- share internet access
- communicate quickly
- access centralized services like databases and cloud storage

---

## 2. Importance of Computer Networks
1. **Resource Sharing**: Networks allow users to share resources such as files, printers, and internet connections.
2. **Communication**: They facilitate communication through various messaging platforms, VoIP, and video conferencing.
3. **Data Management**: Centralized data management is made easier through networked environments.
4. **Scalability**: Networks can easily be scaled to accommodate more users and devices.
5. **Connectivity**: They enable global connectivity, allowing users to connect from anywhere in the world.

### Extra explanation
- **Resource sharing** reduces cost.
- **Communication** becomes faster and more reliable.
- **Centralized management** helps organizations control security and updates.
- **Scalability** means you can expand the network without rebuilding everything.
- **Connectivity** is the reason the internet exists.

---

## 3. Basic Terms You Must Know

### Node
Any device connected to a network is called a node.
Examples:
- computer
- printer
- router
- switch
- smartphone

### Link
The communication path between two nodes is called a link.

### Protocol
A protocol is a set of rules that devices follow when communicating.

### Bandwidth
Bandwidth is the maximum data-carrying capacity of a link.

### Throughput
Throughput is the actual amount of data successfully transferred per second.

### Latency
Latency is the time taken for data to travel from source to destination.

### Jitter
Jitter is the variation in delay over time.

## 8. Common Confusions to Avoid
Bandwidth vs Throughput
Bandwidth = maximum possible capacity
Throughput = actual achieved rate
Latency vs Delay
Latency is the general time taken for data to move
Delay may include transmission, propagation, processing, and queuing components
Protocol vs Standard
Protocol = communication rules
Standard = agreed specification for compatibility
